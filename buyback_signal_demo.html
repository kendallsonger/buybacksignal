Agents
Valuation Agent
Phase 1 ✓
Runs four targeted RAG queries over indexed 10-Q/10-K filings and feeds retrieved chunks to Claude for synthesis. Every claim requires a cited source chunk, enforced by Pydantic schema. Bear case is generated before bull case to guard against confirmation bias.
Insider Signal Agent
Phase 2 ✓
Classifies each Form 4 trade as DISCRETIONARY_BUY, DISCRETIONARY_SELL, PLAN_GOVERNED, or AMBIGUOUS via LLM footnote extraction. Flags short cooling-off periods, plan cancellations, and clusters of discretionary selling before buyback authorizations.
Capacity Check
Phase 3 ✓
Purely deterministic Python — no LLM. Runs four hard gates: blackout window, authorization headroom, leverage ratio, and FCF coverage. Internal DCF and blackout calendar data never enter the model's context window.
Orchestrator
Phase 3 ✓
Combines sub-agent outputs using confidence-weighted scoring (50% valuation, 30% insider, 20% capacity). Maps composite score to action: Initiate ≥0.65, Accelerate ≥0.75, Pause 0.40–0.65, Cancel <0.40.
Design constraints
🔒Internal DCF, FCF forecasts, and blackout calendar are never passed as tokens into any LLM context window — deterministic layer only.
📎Every factual claim cites the exact RAG chunk that supports it. Citation is enforced by Pydantic schema, not just prompting.
⚖️Bear case is structurally required before bull case in all agent prompts — prevents one-sided outputs.
🧑‍💼Permission level is Recommend only. The system documents the case; a human in treasury makes the final call.
📋All outputs are append-only logged for audit committee review.

The problem
A CFO deciding whether to deploy a buyback tranche faces a synthesis problem, not a data problem. The filings, transcripts, and insider-trade records exist — they are scattered across too many sources for any analyst to hold in context at once. Treasury refreshes its view quarterly; dislocations open and close in a week.
The insight
Post-2022 SEC amendments to Rule 10b5-1 collapsed short-cooling-off plan sales from 31% to 1.7% of trades. As genuinely informative insider selling was squeezed out, prices grew less efficient — making discretionary insider buying by CFOs and directors a cleaner conviction signal than at any prior point in the decade.
Build vs. buy
3-year build TCO of $1.23M modestly exceeds a comparable SaaS contract ($1.05M), but no SaaS tool has access to the firm's private DCF, blackout calendar, or authorization headroom — the inputs that constitute the actual moat.
Permission level
The system operates at Recommend — it documents the case and the human decides. The blast radius of a mistimed buyback combined with the jagged error rate of frontier LLMs over messy text rules out autonomous execution at this stage.
